Matte

Matte is a julia library for creating julia-powered dashboards, inspired by Google's material design. Matte makes it easy to convert your existing julia code into a beautiful and powerful dashboard for end users. Matte is easy to use, fast, and powerful.

Please note: Matte is new and is still highly experimental.

Support, bugs, & planned features

General support for help with usage and troubleshooting is best directed to the julialang discourse.

Please report any bugs you find on the github issue tracker. Feature requests can be logged there too.
